## 🏗️ Architecture

### Layout Patterns
- **Auth Pages**: AuthSplitLayout (2-column design)
- **Operational Pages**: OperationalPageShell (header + sidebar + content)

### Component Library
- **GlassPanel**: Glass-effect containers
- **Field/Label/Input**: Form components
- **Button**: Action buttons with variants
- **WorkstationShell**: Layout infrastructure

### Design System
- **Colors**: Primary (#ffb868), Secondary (#b1c5ff), Tertiary (#62df7d)
- **Typography**: 6 levels (display-4xl, page-title, body-md, etc.)
- **Spacing**: 8px, 12px, 20px, 24px, 32px units
- **Icons**: Lucide React & Material Symbols (14px/16px/18px)
